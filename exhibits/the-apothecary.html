What you're looking at. A catalogue of nine things you could build from this stack's five primitives, shelved by how far each sits from code that already runs (hours, days, or weeks). The grading is the honesty: it keeps "what you could build" tied to real effort rather than vague promise.

The Apothecary

Five primitives compose into more than five products. Here are nine, three rigorous, three useful, three strange, each a real thing someone could build. The jars are filled by distance from code that already runs: brimming is hours of work; a few drops is weeks. Tap one to pour it out.

Near, configure / rename · hours
Mid, adapt a seam · days
Far, real new work · weeks+
The tool drawer

The stack this is built from

Every jar below is composed from these five, all public. The point of the catalogue is that you don't have to take "you could build it" on faith: the parts are real and you can read them.

◆ worldweaver the persistent shared world and its residents ◆ the-stable the familiar runtime, where the local minds live ◆ memory-management reusable external memory for any project ◆ review-scheduler a cold reviewer that can't be charmed ◆ prune write it down, build it, prune the living set

Built and run on Linux and Docker.
